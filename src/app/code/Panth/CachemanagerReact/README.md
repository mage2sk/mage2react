# Panth_CachemanagerReact

React/Astro storefront adapter for `Panth_Cachemanager`.

Admin-only cache-warming tool. No storefront integration. This companion exists to keep the Panth React extension suite consistent.

## Sequence

- `Panth_React`
- `Panth_Cachemanager`

## Scope

- Admin/backend-facing only
- No storefront UI, no GraphQL, no Astro components
- Scaffolding-only module
